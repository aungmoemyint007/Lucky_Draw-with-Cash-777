Management System 2.0
Overview

The Management System 2.0 is designed to handle user data and manage cash balances efficiently. This system allows users to log in, create accounts, and update their data with easy access to cash management.

Files

System.cpp: The C++ source code for the management system’s main logic.

System.exe: The compiled executable of the system.

System.o: The object file generated from the C++ source code.

cash.txt: Stores the cash balance data.

userdata.txt: Contains user-related information.

Features

User Login: Allows users to log into the system securely.

Create Account: Users can create a new account and manage their details.

Manage Cash: Provides a function to manage and update the cash balance in real-time.

Flowchart

The following flowchart outlines the system’s operation:

Start: Begin the system.

Display Menu: Users are shown the main menu with options to log in or create an account.

User Login / Create Account: Users can log in or create a new account based on their choice.

Manage Cash: Once logged in, users can manage their cash balance.

Save Data: User data and cash information are saved to userdata.txt and cash.txt.

Exit: The program exits after saving the data.

Usage

Compile System.cpp using a C++ compiler to generate System.exe.

Run the executable to interact with the system.
